# 行为准则

## 我们的承诺

为了营造一个开放、友好的环境，我们作为贡献者和维护者承诺：无论年龄、体型、残疾、民族、性别认同与表达、经验水平、国籍、个人外貌、种族、宗教信仰或性取向，参与本项目和社区的每个人都能获得无骚扰的体验。

## 我们的标准

有助于创造积极环境的行为包括：

- 使用友好、包容的语言
- 尊重不同的观点和经历
- 优雅地接受建设性批评
- 关注对社区最有利的事情
- 对其他社区成员表现出同理心

不可接受的行为包括：

- 使用性化语言或图像，以及不受欢迎的性关注或骚扰
- 挑衅、侮辱/贬损性评论以及人身攻击或政治攻击
- 公开或私下骚扰
- 未经明确许可，发布他人的私人信息，例如实际地址或电子地址
- 在专业环境中可被合理认为不当的其他行为

## 我们的责任

项目维护者有责任阐明可接受行为的标准，并对任何不可接受行为采取适当且公平的纠正措施。

项目维护者有权利和责任删除、编辑或拒绝不符合本行为准则的评论、提交、代码、维基编辑、Issues 及其他贡献，并在适当时封禁任何贡献者。

## 适用范围

本行为准则适用于项目空间内以及个人代表项目或其社区的公共空间。代表项目或社区的情形包括：使用官方项目电子邮件地址、通过官方社交媒体帐户发帖，或在线上/线下活动中担任指定代表。

## 执行

如遇滥用、骚扰或其他不可接受行为，可通过 GitHub Issues 联系项目维护者进行举报。所有投诉都将被审查和调查，并将采取被认为必要且适当的应对措施。项目团队有义务对事件举报者保密。具体执行政策的更多细节可能会单独发布。

不真诚遵守或执行行为准则的项目维护者，可能面临由项目领导层其他成员决定的临时或永久性影响。
